## Design Principles

### 1. **Instant Value**
Every command should return something useful in <1 second

### 2. **Actionable Insights**
Don't just show dataâ€”tell users what to do with it

### 3. **Personal Baselines**
Compare to user's own history, not arbitrary benchmarks

### 4. **Visual Appeal**
Beautiful CLI output creates delight and shareability

### 5. **Progressive Disclosure**
Start simple, allow drilling down for detail

### 6. **Privacy First**
Individual data stays individual by default
